# Uptask

Frontend for a school project on a Trello-style task and project manager

## Tech Stack

- React
- Typescript
- React Query

## Development

### Prerequisites

- [Node.js](https://nodejs.org/)
- [pnpm](https://pnpm.io/)

### Installation

1. Clone the repository:

```bash
git clone https://github.com/oscar370/uptaks_frontend.git
cd uptaks_frontend

```

2. Install dependencies:

```bash
pnpm install

```

3. Start the development server:

```bash
pnpm dev

```

4. Build for production:

```bash
pnpm build

```
